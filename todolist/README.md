# TodoList App

แอปพลิเคชันสำหรับบันทึกรายการงานที่ต้องทำในแต่ละวัน สร้างด้วย React + TypeScript + Vite

## ฟีเจอร์

- ✅ เพิ่มรายการงานใหม่
- ✅ แก้ไขรายการงาน
- ✅ ลบรายการงาน
- ✅ ทำเครื่องหมายว่าเสร็จแล้ว
- ✅ กรองรายการงาน (ทั้งหมด/ยังไม่เสร็จ/เสร็จแล้ว)
- ✅ ธีมมืด/สว่าง

## เทคโนโลยีที่ใช้

- React 19.2.0 พร้อม TypeScript
- Vite สำหรับ development server
- Tailwind CSS สำหรับ styling
- Lucide React สำหรับ icons

## การติดตั้ง

```bash
npm install
```

## การรัน

```bash
npm run dev
```

## การ build

```bash
npm run build
```
